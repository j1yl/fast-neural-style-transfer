# CS190I Final Project: Fast Neural Style Transfer in PyTorch

## Setup

1. Create and activate virtual environment:
```bash
python -m venv .venv
source .venv/bin/activate
```

2. Install dependencies:
```bash
pip install -r requirements.txt
```
